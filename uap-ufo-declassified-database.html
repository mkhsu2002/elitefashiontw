導讀原則

用「檔案館導讀」處理 UAP，而不是用猜測填補空白

本專題將 UAP/UFO 視為官方資料、航安紀錄、感測器資料與歷史檔案來閱讀。每一類資料都盡量標明素材形式、公開機構、官方狀態與可查證連結，讓讀者能分辨哪些是影像、哪些是 PDF、哪些已有官方解析，哪些仍因資料不足而未能歸因。

官方資料優先

以 AARO、NARA、CIA Reading Room、ODNI、DoD/DVIDS 與美國空軍歷史檔為主要來源，避免以二手傳聞取代原始檔案。

案例狀態清楚標示

每個事件至少標出「已解析、未解、分析中、資料不足、歷史檔案」之一，並說明官方判定與剩餘疑問。

素材權限分層判讀

影片、照片、PDF、檔案資訊與政府標章需分開判斷。可引用不一定可商業重製，可下載也不代表可任意改作。

事件分類

事件分類以「來源機構、地區、素材型態、官方狀態」交叉整理。這樣讀者可以先看最新公開影像，也可以回到 Project Blue Book、Roswell 與 CIA Reading Room 這類歷史檔案。

以 AARO Official UAP Imagery 最新新增項目為主，追蹤 PR 系列案例。

2. 中東 / 非洲 / 亞洲案例

以戰區與跨區域軍事平台影像為主，特別標示是否缺乏遙測或多感測器資料。

Africa 2022：未解，熱訊號來源未定 Africa 2024：已解析為候鳥群 Middle East Red Balloon 2024：反光鋁箔氣球 Middle East 2024：未解，缺乏遙測與多模態資料 Middle East Object：不呈現異常行為，但仍未識別 South Asian Object：疑似商業飛機與壓縮假象

3. 美國軍方經典影像案例

以已公開軍方影片、NAVAIR FOIA 影像與 AARO resolution report 為核心。

Go Fast：F/A-18F FLIR 影片，AARO 有解析報告 Gimbal：NAVAIR FOIA 影片，需與 Go Fast、FLIR1 分開查核 FLIR1 / Tic Tac：NAVAIR FOIA 影片，官方未作外星結論 Western U.S. Objects：商業飛機，遠距離與感測器造成誤判 Puerto Rico Case：兩物體近距離飛行，非入水或分裂 Eglin UAP Case：很可能為大型商業照明氣球等輕於空氣物體

4. 歷史檔案案例

以 NARA、CIA Reading Room 與 AARO 歷史報告為主，避免把歷史傳聞和官方檔案混為一談。

Project Blue Book：1947-1969，12,618 件報告，701 件未識別 Roswell：空軍調查指向當時機密氣球計畫相關材料 CIA UFO X-Files：FOIA/CREST 文件與 CIA 歷史研究 KONA BLUE：DHS proposed SAP，AARO 稱未獲批准或正式建立 1940s-1970s files：空軍、情報、總統圖書館與 NARA 多來源

5. 已解析案例

已解析不代表每一個細節都完美復原，而是官方在可用資料下給出最可信歸因。

氣球：PR-004、PR-005、PR-006、PR-009、PR-010、Al Taqaddum、Middle East Red Balloon 鳥群：PR-016、Africa 2024 商業飛機：Western U.S. Objects、South Asian Object 感測器假象：Atmospheric Wake 的尾流影像壓縮 大氣現象與視覺效果：Mt. Etna、parallax、forced perspective

6. 未解案例

未解不是外星證據，而是表示目前資料不夠或尚未歸因。本專題會標明每個未解案例的資料限制。

缺乏遙測資料 缺乏多感測器佐證 只有單一影像或目擊報告 資料品質不足以估算速度、距離或尺寸 AARO 仍可能等待更多資料後更新歸因

Roswell 官方檔案導讀

AARO Historical Record Report 導讀

近年官方回應 off-world technology、reverse engineering 與歷史調查主張的核心報告。

KONA BLUE 與 ORNL 材料分析

把 alleged non-human biologics、外星材料與逆向工程主張拆回 proposed SAP、材料檢測與官方查核結論。

授權引用

授權與引用注意事項

美國政府資料常見 public domain 原則很重要，但不能簡化成「所有東西都可任意商用」。機構標章、第三方權利、肖像、商標、影片平台條款與非背書聲明都要分開判斷。

Public domain 一般原則

美國聯邦政府員工在職務中創作的作品，通常在美國不受著作權保護。但若含第三方素材、承包商素材、商標、標章或個人權利，仍可能有限制。

NARA 資料使用方式

NARA 說明 Catalog 可下載項目可在標示出處下再發布；若頁面標示 donor、copyright 或其他 intellectual property 限制，應只連結或摘要。

AARO / DoD 影像注意事項

DVIDS 要求避免暗示官方背書。商業用途尤其要注意軍種名稱、徽章、制服、機體標誌與第三方權利。

CIA Reading Room 引用方式

CIA.gov 未標 copyright 的資訊通常可重製，但應引用 CIA 為來源，並保留文件標題、Document Number、Release Decision 與 PDF URL。

何時只引用不可重製

遇到第三方版權、照片人物權利、平台無下載選項、影像無明確 public domain 說明，或內容含敏感標章時，優先做摘要與外連。

免責聲明

本頁為繁中導讀與資料索引，不代表 AARO、NARA、CIA、ODNI、DoD 或任何美國政府機構立場，也不暗示官方背書。

事件檔案會收錄哪些資訊

為了讓不同來源的案例能互相比較，每則事件檔案都會保留同一組核心資訊。讀者可以從這些欄位判斷事件來源、素材型態、官方狀態、引用風險與最後查核日期。
