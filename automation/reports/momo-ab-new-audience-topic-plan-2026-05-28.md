## 驗收規則

- 每篇文章完成後需跑 `python3 scripts/content_authenticity_audit.py --slug <slug>`。
- 每篇含 momo 導購連結時，連結需有 `rel="sponsored nofollow"`，文末需有可見導購揭露。
- 每批完成後更新 `automation/momo-brand-recommendation-tracker.csv` 的 `article_created`、`article_slug`、`live_url`、`mention_count`、`last_mentioned_at`。
- 每批完成後跑 `python3 scripts/content_pipeline.py verify`、封面 strict audit、全站 authenticity audit，並透過 GitHub Actions backfill Google Sheets。
