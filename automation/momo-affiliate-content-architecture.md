# momo 聯盟行銷內容分類架構與文章提案

建立日期：2026-05-19

本文件是內部內容規劃檔，用來把 momo 聯盟店家清單轉成 Elite Fashion TW 可長期操作的文章系列、分類歸屬與品牌推薦覆蓋機制。前台文章不可直接揭露本文件中的「主角、配角、覆蓋率、SEO、品牌池、矩陣、批次」等內部規劃語言。

來源檔案：

```text

```

品牌推薦追蹤表：

```text
automation/momo-brand-recommendation-tracker.csv
```

## 一、核心策略

momo 店家不應只用「最高推薦等級」決定是否入文。真正適合本站的做法是：

1. 用 A+ / A 品牌做主推，承擔讀者信任與轉換。
2. 用 B 品牌做配角，擴充商品詞、情境詞、比較維度與替代選項。
3. 用 C 品牌作為背景詞或待查證素材，除非完成商品頁查證，否則不使用強推薦語氣。
4. D 品牌暫緩，不主動置入文章。
5. 每篇文章以讀者問題、生活情境與決策順序為主，不可寫成品牌堆疊。
6. 每篇含導購或商品推薦時，需有可見導購揭露，連結需使用 sponsored / nofollow 屬性。

## 二、品牌角色定義

### 主角品牌

條件：

- 推薦等級 A+ 或高分 A。
- 商品線清楚。
- 與本站既有分類及熟齡女性讀者需求高度吻合。
- 適合放在主要比較表、推薦段落、CTA 或第一層購買判斷。

### 配角品牌

條件：

- 推薦等級 B 或部分中高信心 C。
- 商品詞、場景詞或比較維度有用。
- 可作為替代款、補充類別、不同預算或不同使用情境。
- 不應搶過主角品牌，不使用「最推薦」等強語氣。

### 背景品牌

條件：

- 推薦等級 C 或資訊不足。
- 可用於內部關鍵詞與素材擴充。
- 前台使用前需補查主力 SKU、價格帶、評價數、商品圖與官方背書。

### 暫緩品牌

條件：

- 推薦等級 D。
- 品類不明、信心低、與本站主題不合或無法安全推薦。
- 不進正式文章，除非人工查證後重新分級。

- `article_slug`
- `live_url`
- `mention_count`
- `last_mentioned_at`
- `risk_notes`

推薦分配原則：

1. 每篇文章最多 2 到 4 個主推品牌。
2. 每篇文章可放 3 到 8 個配角品牌，視文章長度與讀者問題而定。
3. 同一品牌連續 3 篇新文都已出現時，下一篇應優先尋找其他可用品牌，除非該篇主題非該品牌不可。
4. 每個小系列至少覆蓋 10 個不同品牌。
5. 每個小系列至少包含 30% 的 B 級配角品牌，以避免所有文章都重複推薦同一批高價值品牌。
6. C 級品牌若要進文，只能放在「補充選項、待比較類型、購買前可再查」的語氣中，並需先查證商品頁。
7. D 級品牌不得進正式文章。

## 五、文章生成與圖片規則

每篇 momo 聯盟文章完成後，需產生同一張圖像內容供：

- 文章封面圖
- OG / Twitter 預覽圖

圖片規格：

- 建議輸出：`1200 x 630`
- 圖像內容應符合該文章情境，不做純抽象背景。
- 人物圖預設以歐美白人與亞洲華人為主，避免不必要的中東、印度、南亞宗教或民族造型。
- 戶外、旅行、居家、咖啡、通勤與工作角落等主題需拉開色調，不可大量重複同一種室內暖光、沙發、筆電桌面構圖。

本地或 Codex 人工產文時，文章完成後應調用 Codex 可用的圖片生成能力產圖。GitHub Actions 自動排程若尚未接入圖片 API，需先標記為 `needs-cover-image`，不得假裝已生成新封面。

## 六、模型品質規則

momo 聯盟文章屬於商業內容與品牌推薦內容，正式撰寫不使用 `gpt-5-mini` 作為預設 writer model。

建議：

- Writer model：`gpt-5.2`
- Planner model：`gpt-5.2`

若因成本或限額需要降級，需在 `latest-run` 或相關紀錄中標明，且該批文章需人工複核語氣、品牌推薦平衡、導購揭露與高風險宣稱。

## 七、分類歸屬規則

新 momo 文章不可新增大分類，應歸入現有分類：

- 戶外、旅行、露營、防曬、防雨、通勤移動：`outdoor-escapes`
- 通勤包款、穿搭、鞋包配件、防曬穿搭：`casual-chic`
- 睡眠、護具、按摩、瑜珈、恢復、旅行疲勞：`wellness-movement`
- 香氛、照明、收納、清潔、咖啡、茶、送禮、居家：`lifestyle-culture`
- 外接螢幕、筆電、拍攝設備、創作者工具、AI 工作角落：`ai-innovation`

若一篇文章跨分類，以讀者的主要搜尋意圖決定分類，而不是以品牌原始 sheet 決定分類。
