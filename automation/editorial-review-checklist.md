# Elite Fashion 自動化文章人工校稿清單

## 發布前必檢
- 標題是否清楚、自然、可搜尋，且與既有文章不高度重複。
- 導言是否直接說明讀者會得到什麼，沒有空話開場。
- 結構是否完整，至少包含問題定義、重點拆解、實作建議、FAQ、CTA。
- 內容是否符合 40+ 熟齡女性與台灣讀者的語境。
- 是否避免浮誇承諾、虛假案例、內部思考過程與 AI 自述。
- 是否有具體情境、步驟、比較點，而不是只有抽象觀點。
- FAQ 是否真正回應搜尋意圖，而不是重複本文段落。
- 是否補上站內延伸閱讀與合理的 CTA。
- `metaTitle`、`metaDescription`、列表標題與列表摘要是否完整。
- 文章是否已寫入前台需要的資料欄位與 Markdown 正文。
- queue、發文紀錄、articles 索引、front listing、search index 是否同步更新。
- 驗證通過後，該篇是否可直接發布到前台。

## 高風險警訊
- 標題只是把舊文換年份。
- 內文沒有台灣場景，只是翻譯腔重寫。
- 全文缺少 FAQ、延伸閱讀、CTA 或 meta 欄位。
- 明明是系列文，卻沒有承接前一篇或下一篇定位。
- 文章已生成，但前台索引、搜尋或 queue 狀態沒有更新。
