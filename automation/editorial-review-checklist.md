# Elite Fashion 自動化文章人工校稿清單

## 發布前必檢
- 標題是否清楚、自然、可搜尋，且與既有文章不高度重複。
- 標題是否避開 `主權協議`、`絕對權威`、`權威之戰`、`絕對主導`、`革命` 等已降權公式。
- 導言是否直接說明讀者會得到什麼，沒有空話開場。
- 結構是否完整，至少包含問題定義、重點拆解、實作建議、FAQ、CTA。
- 內容是否符合 30 到 50 歲白領菁英、高效率生活讀者與台灣讀者的語境。
- 標題、導言、H2、FAQ 與 CTA 是否避免直白使用 `30+`、`40+`、`45+`、`50+`、`熟齡`、`銀髮`、`老人` 等會框定讀者的字眼；除非該篇主題本身就是年齡、醫療、法規或人生階段議題。
- 是否避免浮誇承諾、虛假案例、內部思考過程與 AI 自述。
- 是否有具體情境、步驟、比較點，而不是只有抽象觀點。
- 是否清楚說明適合誰、不適合誰，以及讀者下一步可以怎麼判斷。
- FAQ 是否真正回應搜尋意圖，而不是重複本文段落。
- 是否補上站內延伸閱讀與合理的 CTA。
- `metaTitle`、`metaDescription`、列表標題與列表摘要是否完整。
- 是否沒有使用 `nosnippet`、`data-nosnippet` 或會阻擋搜尋摘要的 robots meta 設定。
- 初始 HTML 是否可直接看到主文、FAQ、延伸閱讀、CTA、導購揭露與頁尾；重要內容不可只靠 JavaScript 動態注入。
- 封面與內容圖片是否有具描述性的 alt text，而不是只重複檔名或空泛標題。
- 文章是否已寫入前台需要的資料欄位與 Markdown 正文。
- queue、發文紀錄、articles 索引、front listing、search index 是否同步更新。
- 驗證通過後，該篇是否可直接發布到前台。

## 分類與策略檢查
- 新文是否優先歸入現行核心分類：`lifestyle-culture`、`wellness-movement`、`casual-chic`、`outdoor-escapes`、`ai-innovation`。
- 是否避免新增 `life-proposals` 或 `high-performance` 類型內容；若題目屬人生提案，應改歸核心分類。
- `wellness-movement` 是否以健康恢復、睡眠、熱感、肌力、步行、身體節奏為主，而不是只寫瑜伽或健身器材。
- `runway-trends` / `designer-perspective` 是否有生活轉譯段落，不能只是概念評論。
- 品牌專案是否使用 `series` 管理，並回到讀者使用情境，不讓品牌名稱取代文章價值。

## 高風險內容檢查
- 健康、醫療、法律、保險、投資、房地產與財務題是否避免保證式結論。
- 是否提供必要的風險提醒，並建議讀者依自身情況諮詢合格專業人士。
- 是否避免捏造醫療效果、投資報酬、法規細節、政府補助、產品價格與專家訪談。
- 產品或品牌文若含合作、贊助或聯盟連結，是否有可見 disclosure，並使用合適的 sponsored / nofollow 屬性。
- Review / Product / FAQ schema 是否能被頁面可見內容支持，不能只為 SEO 加不存在的結構資料。
- 若頁面含影片，是否有可見摘要或逐字稿重點，且 `VideoObject` schema 只標記真實存在的影片內容。
- 是否避免為了在地 SEO 虛構 Google Business Profile、地址、電話、營業時間或實體服務範圍。

## 高風險警訊
- 標題只是把舊文換年份。
- 內文沒有台灣場景，只是翻譯腔重寫。
- 全文缺少 FAQ、延伸閱讀、CTA 或 meta 欄位。
- 明明是系列文，卻沒有承接前一篇或下一篇定位。
- 文章已生成，但前台索引、搜尋或 queue 狀態沒有更新。
- 文章暴露內部策略、選題邏輯、模型提示、審稿規則或自我修正過程。
- 全文沒有任何具體編輯判斷，只是把常識、品牌名或商品詞平鋪直敘。
- 商品、品牌、服務或影片資訊只存在於 schema、JSON 或腳本裡，頁面可見內容看不到。
