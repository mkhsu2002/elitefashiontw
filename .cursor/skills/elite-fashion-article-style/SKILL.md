# Elite Fashion 文章品質規範

## 適用時機

- 撰寫或修改站內任何文章內文（HTML 內 `<article>` / 正文段落）
- 審稿、潤稿、或產出新文章時

## 一、語氣與定位

- **角色**：專業生活時尚編輯，不是行銷文案或 AI 講稿。
- **讀者**：40+ 台灣女性，追求質感與實用並重。
- **語氣**：理性、有顧問感；多用「您」；每段不宜過長，便於手機閱讀。
- **用詞**：台灣慣用語優先——使用「影片」而非「視頻」、「軟體」而非「軟件」、「資訊」而非「信息」。價格標註 TWD 或 USD。

## 二、禁止或極少使用的用語（避免 AI 感與空洞感）

以下詞彙易造成「套路感」或難以理解，**除非真的必要，否則替換或刪除**：

| 避免 | 建議改寫或替換 |
|------|----------------|
| 主權、權威、協議（用在非法律/科技情境） | 選擇權、品味、方法、態度、風格 |
| 深度拆解、深度解析、深度建構 | 一次看懂、從頭說起、實戰做法 |
| 精英、精英階層、精英生活者（每段重複） | 改為「妳」「讀者」「懂得挑選的人」或刪除 |
| 護城河、躍遷、佈署、接口、模組、編碼 | 改用白話：優勢、升級、規劃、管道、單元、邏輯 |
| 分子級、絕對主導、終極裁判權 | 具體描述效果，不堆砌抽象詞 |
| 生命溢價剩餘、地理壓降適應、存有主權 | 改為讀者能懂的說法（例如：多出來的從容、飛行時的舒適、對自己的主導） |
| 權威演進、權威化轉向、權威躍遷 | 演變、轉變、升級 |
| 「這不僅僅是關於 X，這是關於…」 | 開頭改為一句具體情境或問題 |
| 「對於 Elite 讀者而言」反覆出現 | 刪除或改為「若妳…」「實務上…」 |
| 「本文將深度拆解…帶領您領略…助您…」 | 改為簡短導讀或直接進入主題 |
| 「讓我們一同透視…」 | 刪除或改為「以下從…談起。」 |

## 三、結構與條理

- **標題**：清楚、可讀，避免過長副標堆砌關鍵字。可保留一個副標說明角度即可。
- **導言**：1～3 句點出「為什麼要讀這篇」或一個具體情境，不要整段術語。
- **小標（h2/h3）**：依邏輯順序（例如：概念 → 實作 → 在地建議 → 總結），編號若使用則一致（一、二、三 或 1. 2. 3.），且**不要出現兩個相同編號**（如兩個「二、」）。
- **段落**：每段 2～4 句為宜，必要時用條列（ul/ol）或短 bullet，方便掃讀。
- **結尾**：收束在具體建議或一句有餘韻的話，避免重複口號型句子（如「Elite Fashion 邀請您…絕對主權」）。

## 四、文法與格式

- **語體**：全文繁體中文，標點用全形。
- **錯字／用詞**：常見需修正——「職是」→「而是」；「極極致」→「極致」；「債權人」在比喻時多為誤用，改為「籌碼」「武器」等。
- **英文**：專有名詞（品牌、型號、學名）可保留；抽象概念優先中文，若用英文則括號註解一次即可，不要整段中英夾雜。
- **HTML**：正文中不用 Markdown 語法（`**粗體**` 在 HTML 中不會顯示），改用 `<strong>` 或 `<b>`。所有 `<p>`、`<div>` 必須正確閉合。
- **重複字**：若使用首字放大（drop-cap），注意下一字不要與首字重複成「在在」「從從」等，可改寫句首。

## 五、在地化

- **地名與場景**：可保留台北、信義區、松山機場、天母等，但不必每段都提「台北精英」。
- **社群與平台**：PTT、Dcard 等可提及，作為情境或參考即可，不堆砌「PTT 某某版討論」。
- **品牌與通路**：台灣買得到的品牌、百貨、電商可具體寫；代購、價格以 TWD 或實務說法為主。

## 六、改寫範例（Before → After）

**Before（過度術語）：**  
「在 2026 年，飛行穿搭已完成了從舒適休閒到『移動主權協議』的權威演進。真正的全球主導者追求能將跨洋飛行轉化為生命溢價剩餘、具備移動權威的深度接入。」

**After（編輯口吻）：**  
「長途飛行穿什麼，會直接影響落地後的精神與狀態。與其隨便一件寬褲，不如選一條耐坐、好動、看起來仍得體的長褲——差別就在這裡。」

---

**Before（空洞結尾）：**  
「Elite Fashion 邀請您，在這場關於味覺的覺醒運動中，優雅領銜。當您擁有一份能辨識風土的美感時，您擁有的其實是那份專屬於精英者的、跨越地域與文化的絕對主權。」

**After（具體收束）：**  
「從產區、烘焙到沖煮，每一環節都是選擇。下次走進咖啡館，不妨先問一句豆子來自哪裡——妳的味覺地圖，就從這一問開始擴張。」

## 七、產出前自檢清單

- [ ] 標題與導言沒有堆砌「主權／權威／協議／深度拆解」等詞
- [ ] 沒有重複「對於 Elite 讀者而言」「本文將深度拆解…」
- [ ] 小標編號連續且不重複，段落長度適中
- [ ] 使用「您」、台灣用語（影片／軟體／資訊）、必要時 TWD
- [ ] HTML 內無 `**粗體**`，標籤皆正確閉合
- [ ] 結尾為具體建議或一句收束，而非口號重複
